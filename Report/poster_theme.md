# Poster Theme — "Deep Plum + Sage"

A deliberately different aesthetic from the reference's red+navy. Academic but warmer, modern, and clearly distinguishable while keeping the same section grid.

## Palette

| Role | Hex | Use |
|------|-----|-----|
| **Primary (dark)** | `#2E1A47` | Title band, section-header pills, body headings, borders |
| **Primary accent** | `#6B4E8C` | Secondary headings, accent lines, stat numbers |
| **Bright accent** | `#F4A261` | Highlight numbers ("86%"), section-header icons, call-outs |
| **Supporting** | `#86A88E` | Sub-accents, bullet markers, hover / secondary pills |
| **Background** | `#FAF7F0` | Poster canvas (warm cream, not pure white) |
| **Section card** | `#FFFFFF` | Content cards, with 6% grey border |
| **Text primary** | `#1E1E2E` | Body text |
| **Text muted** | `#5D5F76` | Captions, labels, registration numbers |

## Typography

All fonts are common Windows / macOS system fonts so the poster renders consistently on any reviewer's machine.

| Role | Font |
|------|------|
| Poster title | **Segoe UI Black**, uppercase, 48–56 pt |
| Section headers | **Segoe UI Semibold**, uppercase, 18–22 pt |
| Body text | **Segoe UI**, regular, 11–13 pt |
| Stat numbers | **Cascadia Mono Semibold** (tech feel), 28–44 pt |
| Captions & labels | **Segoe UI Light**, 9–10 pt |

Fallback: Calibri for all Segoe UI weights; Consolas for Cascadia Mono.

## Visual motifs

- Section headers are placed in **solid plum pill bars** (4% corner rounding) with white text.
- Each content card sits on the cream canvas with a **subtle 1 pt grey border** at 6% opacity.
- Stat numbers appear in **Cascadia Mono** to evoke the tech / code theme of the project.
- Bullet markers are **small sage-green triangles** pointing right.
- The institute header strip keeps the plum primary (not the reference's red), making the distinction immediate.
